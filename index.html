Raccourcis

Commande Déscription
Ctrl + A Aller au début de la ligne que vous êtes en train de taper.
Ctrl + E Allez à la fin de la ligne que vous êtes en train de taper.
Ctrl + L Effacer l'écran.
Ctrl + U Efface la ligne avant le position du curseur si vous êtes a la fin Efface la ligne entiére.
Ctrl + C Stop tout se que vous faites.
Ctrl + W Supprime le mot avant le curseur.
Ctrl + K supprime la ligne apres le curseur.
Ctrl + T Echange les deux caractéres avant le curseur.
Ctrl + F Déplace un caractére vers la gauche.
Ctrl + B Déplace un caractére vers la droite.
Esc + F Déplace un mot vers la gauche.
Esc + B Déplace un mot vers la droite.
Esc + T Echanger les deux mots avant le curseur.
Tab Compléte automatiquement les dossiers et les fichiers.

Commande principale

Commande Descritpion
cd Aller sur le répertoire d'accueil
cd - Retourner sur le répertoire précédant
ls Faire apparaitre une liste courte
ls -l Faire aparaitre une liste longue
ls -a liste tout les fichier ainsi que les cachés
ls -lh Longue liste avec la taille des fichers
ls -R Affiche tout le contenu du dossier récursive
sudo "command" Exécuter la commande avec les privilèges de sécurité du superutilisateur
open "file" ouvre un fichier
top Affiche les processus actifs . Appuyez sur q pour quitter.
nano "file" Ouvre le fichier en utilisant l'éditeur de nano
vim "file" Ouvre le fichier en utilisant l'éditeur de vim
clear nettoyer l'écran
reset Remet l'affichage du terminal

Commande Description

!"value" Exécute la dernière commande tapée qui commence par «valeur»
!! Exécutez la dernière commande tapée

Chercher

Commande Descritpion
find [dir] -name "search_pattern" Rechercher des fichiers , par exemple trouver / " file.txt " -name utilisateurs
grep "search_pattern" "file" Rechercher toutes les lignes qui contiennent le motif, par exemple grep " Tom " file.txt
grep -r "search_pattern" "file" rechercher toutes les lignes qui ne contiennent pas le motif "."
grep -v "search_pattern" "file" Rechercher toutes les lignes qui ne contiennent pas le motif
